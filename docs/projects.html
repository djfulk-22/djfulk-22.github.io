UNC Baseball Attendance Prediction

Collaborative MADS capstone project with Adam Cooke and Arvind Madan for UNC Athletics. Built an end-to-end attendance forecasting framework and Streamlit decision-support application for UNC baseball home games using historical attendance, opponent context, scheduling features, weather, academic calendar timing, Google Trends, promotions, and team performance data.

CourtSpace 3D: Interactive NBA Shot Visualization App

Collaborative Streamlit app with Arvind Madan that visualizes NBA player shot attempts as 3D trajectories on a realistic half court. The app uses NBA shot chart data, Plotly 3D rendering, contextual filters, and a league-relative hot/cold zone heatmap to help users explore player shooting patterns across seasons, opponents, shot types, venues, and game periods.

My role: Created the first working prototype, implemented the initial Streamlit app, built the core 3D shot visualization pipeline, added hover text metadata, quarter filtering, hot/cold heatmap mode, the heatmap scaling slider, and designed the caching system for NBA API requests.

Skills shown: Python, Streamlit, Plotly, nba_api, sports analytics, 3D visualization, data filtering, API caching, interactive dashboard development

Pitch Forecaster: MLB Pitch Type Prediction

Collaborative machine learning project with Arvind Madan and Elliott Walker focused on predicting the next MLB pitch type from pitch-level baseball data. The project used public Statcast/Baseball Savant data, game-context variables, pitch sequencing features, pitcher and batter tendencies, and multiclass classification models to explore how historical pitch patterns can be used to forecast future pitch selection.

Project work: Developed a machine learning pipeline that cleaned and transformed pitch-level data, engineered lagged sequence features, enriched observations with pitcher and batter arsenal tendencies, and evaluated multiple modeling approaches including multinomial logistic regression, random forest, neural network, and ensemble models. The final ensemble model balanced overall accuracy with stronger class-wise performance across imbalanced pitch types.

Skills shown: Python, pandas, scikit-learn, TensorFlow/Keras, Optuna, sports analytics, feature engineering, multiclass classification, neural networks, ensemble modeling, model evaluation
